# Installation

## Quick Install

Run this command in your terminal:

```bash
curl -L https://github.com/pxdim/project-doc-control-skill/raw/main/project-doc-control.skill -o ~/.claude/skills/project-doc-control.skill
```

Then restart Claude Code.

## Manual Install

1. Download `project-doc-control.skill` from the [Releases](https://github.com/pxdim/project-doc-control-skill/releases) page
2. Copy to `~/.claude/skills/` directory
3. Restart Claude Code

## Verify Installation

After restarting, you should see `project-doc-control` in your available skills list.
